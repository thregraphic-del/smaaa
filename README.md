## هيكل أوراق Google Sheets

### 1. ورقة `المبيعات` — عمود لكل معلومة

| العمود | اسم الرأس (Header) | المحتوى | مثال |
|--------|-------------------|---------|------|
| A | `تاريخ_البيع` | تاريخ إتمام البيع | `2025-06-15` أو `15/06/2025` |
| B | `اسم_الفرع` | اسم الفرع | `جدة` / `الرياض` / `أبها` / `الدمام` |
| C | `الماركة` | ماركة السيارة | `تويوتا` / `لكزس` / `هيونداي` / `نيسان` |
| D | `الموديل` | موديل السيارة | `كامري` |
| E | `السنة` | سنة التصنيع | `2024` |
| F | `سعر_البيع` | سعر البيع بالريال (رقم كامل) | `95000` |
| G | `التكلفة` | تكلفة الشراء بالريال | `82000` |
| H | `الأرباح` | الربح الصافي (يمكن أن تكون معادلة `=F2-G2`) | `13000` |
| I | `طريقة_الدفع` | طريقة الدفع | `نقد` / `تمويل` / `بطاقة` |
| J | `عميل_VIP` | هل العميل VIP؟ | `نعم` / `لا` أو `TRUE` / `FALSE` |
| K | `طلب_تمويل` | هل قدّم طلب تمويل؟ | `نعم` / `لا` |
| L | `أيام_في_المخزون` | كم يوماً كانت السيارة بالمخزون قبل البيع | `45` |

### 2. ورقة `المخزون` — السيارات الموجودة حالياً

| العمود | اسم الرأس (Header) | المحتوى | مثال |
|--------|-------------------|---------|------|
| A | `تاريخ_الدخول` | تاريخ إدخال السيارة للمخزون | `2025-05-01` |
| B | `الماركة` | ماركة السيارة | `تويوتا` |
| C | `اسم_الفرع` | الفرع المخصص له | `جدة` |
| D | `الحالة` | حالة السيارة | `متاح` / `محجوز` / `مباع` |
| E | `أيام_في_المخزون` | عدد أيام في المخزون (ضع المعادلة: `=TODAY()-A2`) | `52` |
| F | `طريقة_الدفع` | (اختياري) | `تمويل` |

## خطوات النشر (Deployment)

1. افتح Google Sheet → **Extensions → Apps Script**
2. احذف أي كود موجود وألصق محتوى `Code.gs` بالكامل
3. احفظ (Ctrl+S)
4. من القائمة العلوية: **Deploy → New deployment**
5. اختر النوع: **Web App**
6. اضبط الإعدادات:
   - **Execute as:** Me
   - **Who has access:** Anyone
7. انقر **Deploy** وانسخ الـ URL الظاهر
8. افتح `index.html` وعدّل متغير `API_URL` ليصبح الـ URL الجديد

```javascript
const API_URL = 'https://script.google.com/macros/s/YOUR_DEPLOYMENT_ID/exec';
```

## عتبات صحة المخزون

يمكن تعديل هذه القيم في أعلى ملف `Code.gs`:

```javascript
const THRESHOLDS = { NEEDS: 60, STALE: 90 };
// NEEDS  → سيارة أمضت أكثر من 60 يوماً (يحتاج متابعة)
// STALE  → سيارة أمضت أكثر من 90 يوماً (راكد)
```
